Movie Booking Application

GitHub Live Demo

Built a full-stack movie booking application enabling users to register, login, search movies, and book tickets with role-based authentication. Admin dashboard supports ticket management and seat availability tracking.

Core Features

  • User Registration, Login & Password Reset
  • Browse & Search Released Movies
  • Movie Ticket Booking with Seat Selection
  • Admin Dashboard for Ticket Management
  • JWT Authentication & Role-Based Authorization
  • Validation Handling & Exception Management
  • Logging, Monitoring & Session Management
  • DTO Mapping & Lombok Integration
  • Auto SOLD_OUT Trigger for Ticket Availability

Tech Stack

Java Spring Boot Angular JWT MySQL JPA Hibernate REST APIs Lombok JUnit Mockito Git

Hotel Management System

GitHub Live Demo

Developed a microservices-based Hotel Reservation System with RESTful APIs for room search, booking, reservations, and cancellations. Integrated Kafka for asynchronous communication and Swagger for API documentation.

Modules & APIs

  • Room Search API for available room discovery
  • Room Booking API with customer & reservation details
  • Reservation Details API using reservation number
  • Reservation Cancellation API with room release workflow
  • Kafka-based asynchronous event processing
  • Swagger Integration for API documentation
  • Unit Testing using JUnit & Mockito
  • Microservices Architecture Implementation

Tech Stack

Java Spring Boot React Microservices Kafka Swagger REST APIs JUnit Mockito Docker Git
